# zml-games-collection
Collection of small games written by oneself

# 说明
全是自己用vue写的小游戏，提高代码以及逻辑能力，娱乐中学习，让学习不干燥！

广告小游戏合集，主要存放平常在手机上刷到的广告小游戏，不是百分百还原，主要简单的实现游戏原理以及玩法

### 2. zml-game-bird
管道小鸟

### 3. zml-game-gobang
人机五子棋

### 4. zml-game-pacman
吃豆人 - H5版

### 5. zml-game-pacman-canvas
吃豆人 - canvas版

### 6. zml-game-snake
